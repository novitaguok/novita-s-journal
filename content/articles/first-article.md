Welcome to my new blog powered by Hashnode and Next.js!

When I publish an article on Hashnode, it automatically commits a file just like this one directly to my GitHub repository. Next.js reads this file and statically builds my website!

## Why this is awesome
1. I get a green square on my GitHub contribution graph.
2. I get to write in Hashnode's beautiful editor.
3. I keep my custom Next.js design.
4. Users can comment at the bottom of the page via Giscus!
